My Work

Code-Jump様の模写課題を、基本的な静的ページからJavaScriptを含む上級者向け課題まで全て制作しました。単なる模写ではなく、見やすさやコードの整合性を意識して取り組んでいます。

Cofe Shop Menu

マウスオーバー時のアニメーションやページ内リンク、背景の固定表示など、基本的なコーディングを学習しました。見本に忠実に再現しつつ、テキストシャドウなどで視認性を向上させています。

Sneaker Brand PV

外部ライブラリを用いたスライダーを実装し、ハンバーガーメニューやグリッドレイアウトなど、モダンUIの基礎やレスポンシブデザインを習得しました。

Flower Gallery LP

特定区間のみメニューや背景が登場するアニメーションを実装しました。スクロールポジションの概念が難しかったため、数値化ツールを自作することで挙動を理解しました。

English School LP

キーフレームを用いたスライドショーを実装し、画像の切替タイミングや調整方法を学びました。また、外部プラグインによるフェードインアニメーションを実現しました。

UI Design

UIやスキル紹介のアイコンなど、当初はテンプレート的な見た目で、世界観を十分に表現できていませんでした。

UI Design

User Experience

制作物サイトのサムネイルは、デザインは凝っていたものの、模写サイトのため印象が薄くなっていました。

User Experience

マウスオーバーでサイト全体図が画面いっぱいに展開するよう再設計。模写サイトの弱さを演出で補い、ユーザー体験を向上させました。

Visual Detail
